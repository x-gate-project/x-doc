# 铸造

## 铸造（Mint）的使用方法

1. 在“铸造”界面中，从“Mint Using”的下拉菜单中选择要锁定的代币，从“Made”的下拉菜单中选择发行目标网络。
    
    ![Mint_4.png](/img/docs/Mint_2.png)
    
2. 在“Mint Using”的数量输入框中输入想要铸造的数量。输入后，“Made”的数量显示框中会显示相同的数值。
    
    ![image.png](/img/docs/Mint_3.png)
    
3. 点击“Mint”按钮。点击后，钱包将弹出显示，检查手续费等显示内容，并确认交易。
4. 当交易在区块链上处理完成后，将会反映在您的钱包中。由于区块链的网络拥堵情况，处理完成可能需要数秒到数十分钟不等。
