# X-Swap란 무엇인가?

X-Swap은 서로 다른 네트워크 상의 **스테이블 코인(USDTX, USDCX)을 스왑(교환)** 할 수 있는 기능입니다.

이를 통해 사용자는 네트워크 간 유동성을 확보하고, 자산 관리를 보다 자유롭게 할 수 있습니다.

### **지원하는 네트워크**

X-Swap은 X-Gate와 마찬가지로 **5개의 주요 블록체인** 을 지원합니다.

- 이더리움(Ethereum)
- Japan Open Chain
- 아비트럼(Arbitrum)
- 베이스(Base)
- 아발란체(Avalanche)
